📤 מה להעלות לקלוד בשיחה חדשה

בכל שיחה חדשה — קלוד לא זוכר כלום. הנה מה שצריך להעלות לפי המשימה:

המשימהקבצים להעלות
עדכון עיצוב כללי (צבע, גופן, spacing)cs-v1.html
עדכון תוכן About (טקסט, קישורים)about.html
עדכון דף בית (סדר קארדים, כותרות)index.html
עדכון פרויקט ספציפיהקובץ הספציפי (למשל agents.html)

הוספת פרויקט חדשcs-v1.html + index.html

🃏 ניהול קארדים — כותרות, סדר ורקע

הקובץ cards-content.txt שולט על כל הקארדים בדף הבית. כל שינוי — כותרת, תת-כותרת, סדר — נעשה שם.

שינוי כותרת / תת-כותרת של קארד

ערכי את השדות הרלוונטיים ב-cards-content.txt:

CARD 1
------
File: agents.html
Title: Autonomous agents          ← שני את זה
Subtitle: AI · FinOps · PointFive  ← ואת זה
Card background color: #E8EDF2
אחרי העריכה: העלי cards-content.txt + index.html לקלוד ותגידי: "עדכן את index.html לפי cards-content.txt"

שינוי סדר הקארדים

שני את המספרים ב-cards-content.txt — CARD 1 יהיה תמיד ראשון בגריד:

לדוגמה — להעביר DSPM למקום ראשון:

CARD 1
File: dspm.html
Title: Cloud data security
...

CARD 2
File: agents.html
Title: Autonomous agents
...
חשוב: אחרי שינוי סדר — עדכני גם את רשימת ה-PROJECTS ב-JS של כל קובץ פרויקט (ראי סעיף "ניווט בין פרויקטים" למטה), אחרת כפתורי Next/Previous יתבלבלו.

שינוי צבע רקע קארד

שני את Card background color ב-cards-content.txt לכל hex color שתרצי:

Card background color: #F2EFE6  ← כל צבע hex
